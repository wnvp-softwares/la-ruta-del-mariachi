# la-ruta-del-mariachi
Sistema y proyecto de aporte cultural y generacion de rutas para el turismo en Cocula, Jalisco; Cuna Mundial del Mariachi

## El sistema estará dividido en 3:
- Servidor
**Donde toda la informacion, programacion y lanzamiento del servidor se contendrá**
- Cliente
**Donde el cliente web (pagina web) de acceso publico se contendrá**
- App
**Donde la app movil se desarrollara (con Expo)**
